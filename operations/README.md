# Operations

Esta pasta concentra runbooks e rotinas operacionais do projeto.

Use este espaço para registrar:

- procedimentos de manutenção;
- rotinas de atualização;
- comandos de suporte;
- políticas operacionais;
- troubleshooting.

Exemplos:

- reindexação do RAG;
- refresh de legislação;
- validação de embeddings;
- procedimentos de deploy ou integração.
