# Workflows

Esta pasta reúne fluxos de trabalho multi-etapas do sistema.

Use este espaço para documentar:

- sequências operacionais;
- handoffs entre agentes e subagentes;
- checkpoints humanos;
- critérios de sucesso e fallback.

Exemplos:

- atendimento inicial;
- triagem jurídica;
- geração de resposta institucional;
- ingestão e atualização de base normativa.
