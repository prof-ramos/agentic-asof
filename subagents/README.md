# Subagents

Esta pasta concentra subagentes especializados que podem ser orquestrados pelos agentes principais.

Exemplos de uso:

- subagente jurídico;
- subagente de classificação documental;
- subagente de triagem de e-mails;
- subagente de análise de fluxos internos.

Convenção sugerida:

- um subdiretório por subagente;
- um `README.md` com papel, gatilhos de uso, contexto mínimo e formato de resposta.
