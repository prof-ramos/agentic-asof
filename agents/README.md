# Agents

Esta pasta concentra a definição dos agentes principais do projeto.

Use este espaço para registrar:

- escopo de cada agente;
- responsabilidades e limites;
- interfaces esperadas;
- dependências com `RAG/`, `workflows/` e `operations/`.

Convenção sugerida:

- um subdiretório por agente principal;
- um `README.md` por agente descrevendo objetivo, entradas, saídas e ferramentas.
